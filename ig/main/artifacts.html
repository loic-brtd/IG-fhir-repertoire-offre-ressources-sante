This page provides a list of the FHIR artifacts defined as part of this implementation guide.

Comportement: Capability Statements

La ressource CapabilityStatement permet de définir les capacités spécifiques que les systèmes doivent supporter pour être conforme à cet Implementation Guide (IG). Les systèmes conformes à cet IG doivent déclarer leur conformance à au moins une de ces ressources CapabilityStatement.

ror-consommateur

Système qui est autorisé à consommer les données du ROR.
ror-serveur

Système ROR National Le Répertoire Opérationnel des Ressources (ROR) est le gisement de données qui décrit les offres de santé au niveau national.

Comportement: Search Parameters

La ressource SearchParameter permet de définir de nouveaux paramètres de recherche pour les serveurs RESTful. Ces paramètres de recherche peuvent également être utilisés pour trier ou inclure des ressources reliées.

ror-sp-CapacityType

Paramètre de recherche pour récupérer la nature de la capacité.
ror-sp-additional-bed-type

Paramètre de recherche pour récupérer les types des lits disponnibles
ror-sp-capacity-closing-type

Paramètre de recherche pour récupérer le type de la fermeture de la capacité
ror-sp-capacity-status

Paramètre de recherche pour récupérer le statut de la capacité.
ror-sp-capacity-update-date

Paramètre de recherche pour récupérer la date à jour de la capacité
ror-sp-crisis-type

Paramètre de recherche pour récupérer le type de la crise
ror-sp-equipement-type

Paramètre de recherche pour récupérer le type d’un equipement.
ror-sp-equipment-feature

Paramètre de recherche pour récupérer le type de caractéristique d’un equipement.
ror-sp-gender-capacity-available

Paramètre de recherche pour récupérer le genre disponnible de la capacité.
ror-sp-limit-value

Paramètre de recherche pour récupérer la valeur limite
ror-sp-location-near-insee-code

Paramètre de recherche similaire au paramètre spécial “near”, à l’exception que celui-ci pointe sur l’adresse et non pas la position géographique
ror-sp-nb-capacity

Paramètre de recherche pour récupérer le nombre de la capacité.
ror-sp-residential-number

Paramètre de recherche pour récupérer le nombre d’habitation
ror-sp-residential-type

Paramètre de recherche pour récupérer le type d’habitation
ror-sp-temporality-capacity

Paramètre de recherche pour récupérer les affectations temporaires
ror-sp-temporary-assignement

Paramètre de recherche pour récupérer les affectations temporaires
